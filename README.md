# DevConnect API

DevConnect API is a backend service designed to facilitate developer collaboration by enabling users to create projects, discover opportunities, and connect with other developers.

The platform provides secure authentication, project management, collaboration requests, bookmarking, and discussion capabilities through a RESTful API architecture.

## Features

* JWT-based Authentication & Authorization
* User Profile Management
* Project Creation and Management
* Collaboration Request System
* Project Comments and Discussions
* Bookmarking Functionality
* Search, Filtering, and Pagination
* Role-based Access Control

## Tech Stack

* FastAPI
* SQLAlchemy
* MySQL
* Pydantic
* JWT Authentication

> 🚧 This project is currently under active development. Features, API contracts, and documentation are subject to change as development progresses.
